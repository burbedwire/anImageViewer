# ImageViewer

Windows用画像ビューワーアプリケーション。CustomTkinterとPillowを使用して構築されています。

## プロジェクト概要

ImageViewerは、Windows上で動作する軽量な画像ビューワーアプリケーションです。画像の表示だけでなく、クリップ、解像度変換、透過色指定、一括変換などの画像編集機能を提供します。

### 主な機能

- **画像表示**: ファイルダイアログまたはドラッグアンドドロップで画像を開く
- **フォルダ内ナビゲーション**: 矢印キーで同一フォルダ内の画像を前後に移動
- **クリップ機能**: 画像の一部を切り出して保存
- **解像度変換**: 画像のリサイズと形式変換（JPG/PNG）
- **透過色指定**: 背景色を自動検出して透過PNGとして保存
- **一括変換**: 複数の画像をまとめて変換

## 環境

- **OS**: Windows 10/11
- **言語**: Python 3.14
- **仮想環境**: venv

### 依存ライブラリ

| ライブラリ | 用途 |
|-----------|------|
| customtkinter | モダンなGUIコンポーネント |
| Pillow | 画像処理 |
| tkinterdnd2 | ドラッグアンドドロップ対応 |

## 起動方法

### 1. 仮想環境の作成と有効化

```bash
cd src
python -m venv venv
venv\Scripts\activate
```

### 2. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 3. アプリケーションの起動

```bash
python main.py
```

## 使用方法

### 画像を開く

- **メニュー**: ファイル → 開く（Ctrl+O）
- **ドラッグアンドドロップ**: 画像ファイルをウィンドウにドロップ

### フォルダ内ナビゲーション

| キー | 動作 |
|-----|------|
| ← | 前の画像 |
| → | 次の画像 |
| ↑ | 1枚目の画像 |
| ↓ | 最後の画像 |

### クリップ機能

1. メニュー: 編集 → クリップ（Ctrl+C）
2. 画像上でドラッグしてクリップ領域を選択
3. 「クリップして保存」ボタンを押す
4. 保存先と形式（PNG/JPG）を選択

### 解像度変換

1. メニュー: 変換 → 解像度変換（Ctrl+R）
2. 変換モードを選択（比率指定 / 幅指定 / 高さ指定）
3. 保存形式（JPG/PNG）を選択
4. JPGの場合はQualityを指定（1-100）
5. 「変換して保存」ボタンを押す

### 透過色指定

1. メニュー: 編集 → 透過色指定（Ctrl+T）
2. 画像をクリックして背景色を選択
3. 許容範囲スライダーで透過の許容誤差を調整（0-100）
4. 「マスクプレビュー」で透過領域を確認（赤オーバーレイ点滅）
5. 「透過して保存」ボタンを押してPNGとして保存

### 一括変換

1. メニュー: 変換 → 一括変換（Ctrl+Shift+R）
2. 「画像を選択」で変換対象画像を複数選択
3. 「出力フォルダを選択」で保存先を指定
4. 解像度変更モードと変換パラメータを設定
5. 「一括変換実行」ボタンを押す

## サポートされる画像形式

- JPG / JPEG
- PNG
- BMP
- GIF
- TIFF
- WebP
