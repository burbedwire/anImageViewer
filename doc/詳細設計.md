# 詳細設計書

## 1. 概要

### 1.1 製品名
ImageViewer

### 1.2 製品概要
Windows上で動作する画像ビューワーアプリケーション。画像の表示、クリップ、解像度変換、透過色指定、一括変換の機能を提供する。

### 1.3 開発環境

| 項目 | 内容 |
|-----|------|
| OS | Windows 10/11 |
| 言語 | Python 3.14 |
| 仮想環境 | venv |
| IDE | Visual Studio Code |

## 2. 技術構成

### 2.1 使用ライブラリ

| ライブラリ | バージョン | 用途 |
|-----------|-----------|------|
| customtkinter | 最新版 | モダンなGUIコンポーネント（テーマ対応） |
| Pillow | 最新版 | 画像の読み込み、加工、保存 |
| tkinterdnd2 | 最新版 | ドラッグアンドドロップ機能 |
| pytest | 最新版 | テストフレームワーク |
| pyinstaller | 最新版 | 単一EXEへのパッケージング |

### 2.2 アーキテクチャ

アプリケーションは以下の層で構成される：

```
┌─────────────────────────────────────────┐
│           プレゼンテーション層             │
│  (ImageViewerApp, Dialogクラス群)         │
├─────────────────────────────────────────┤
│           ビジネスロジック層               │
│  (画像処理、マスク計算、サイズ計算)          │
├─────────────────────────────────────────┤
│             データアクセス層               │
│  (Pillowによる画像I/O)                   │
└─────────────────────────────────────────┘
```

### 2.3 設計思想

- **シンプルさ**: 単一ファイルに機能をまとめることで、保守性を高める
- **非破壊編集**: 元の画像は変更せず、新規ファイルとして保存する
- **ユーザーフレンドリー**: ドラッグアンドドロップ、キーボードショートカットによる直感的な操作
- **レスポンシブ表示**: ウィンドウサイズに合わせて画像を縮小表示

### 3.2 クラス構成

#### 3.2.1 ImageViewerApp (app.py)

画像ビューワーのメインウィンドウクラス。

**継承**: `ctk.CTk`

**主要属性**:
- `_current_image_path: Optional[Path]` - 現在開いている画像パス
- `_image_list: List[Path]` - 現在表示中のフォルダの画像リスト
- `_current_index: int` - 現在の画像インデックス
- `_current_image: Optional[Image.Image]` - PIL Imageオブジェクト
- `_photo_image: Optional[ImageTk.PhotoImage]` - キャンバス表示用のPhotoImage
- `_canvas_image_item: Optional[int]` - キャンバス上の画像アイテムID

**主要メソッド**:

| メソッド | 説明 |
|---------|------|
| `open_image(file_path: str) -> bool` | 画像ファイルを開く |
| `_display_image(img: Image.Image) -> None` | キャンバスに画像を表示 |
| `_prev_image(event: tk.Event) -> None` | 前の画像を表示 |
| `_next_image(event: tk.Event) -> None` | 次の画像を表示 |
| `_first_image(event: tk.Event) -> None` | 1枚目の画像を表示 |
